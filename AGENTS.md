# Instruções para o Codex - Site Universe

Responda sempre em português do Brasil.

Este projeto é o Site Universe.

Este módulo atual é somente o frontend inicial:
- Home
- Login
- Registro
- Verificação de e-mail
- Recuperação de senha
- Redefinição de senha

Não implementar pagamento.
Não implementar recompensas.
Não implementar banco de dados.
Não implementar lógica sensível no frontend.
Não usar localStorage como segurança real.
Não fazer hash de senha no frontend.
Não enviar e-mail diretamente pelo frontend.

Explique planos, progresso, erros e arquivos alterados em português.

Comandos, nomes de arquivos, nomes de bibliotecas, mensagens do terminal e código podem permanecer em inglês quando fizer sentido.
